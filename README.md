# 📚 StudyTogether – AI-Powered Study Collaboration System

A modern Progressive Web App (PWA) designed to help university students collaborate smarter using real-time tools, AI-powered tutoring, and gamification — all in one platform.

## ✨ Features

- 🔐 **Google OAuth Login** with Firebase Auth
- 🤝 **Smart Group Matching** based on course & schedule
- ✅ **Kanban Task Board** for real-time collaboration
- ✍️ **Live Notes Editor** for shared group notes
- 🤖 **AI Chatbot Tutor** (Hugging Face.js) for STEM & code questions
- 🎯 **Gamification:** badges, leaderboards, and streaks
- 📊 **Teacher/Admin Dashboard** with anonymized analytics
- 📱 Fully responsive design (Mobile, Tablet, Desktop)

---

## 🧠 AI-Powered Assistance

- Ask academic questions via chatbot
- Use **voice input** to chat hands-free
- Get **AI-generated summaries** of discussions
- 📌 Coming soon: Flashcards, auto-task suggestions, and more!

## 🧩 Project Architecture

- **Component-Based Structure** for reusability
- **Layouts:** AuthLayout, MainLayout with Sidebar
- **Modular Pages:** Home, Dashboard, StudyGroup, Chat, Notes
- **Realtime Firebase Integration** for tasks, notes, messages
- **Class Diagram:**  
  `User`, `StudyGroup`, `Task`, `AIResponse`, `Note`, `Message`

## 🖥️ Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/study-together.git
cd study-together

# Install dependencies
npm install

# Start dev server
npm run dev
```
